# Customer_Shopping_Behavior_Analysis
Analyzed customer purchase data using SQL, Python, and Power BI to identify buying patterns, segment customers, and track revenue trends. Built an interactive dashboard to visualize insights on purchase frequency, age-based spending, and shipping preferences.

## Overview
This project analyzes customer shopping behavior to identify purchase patterns, customer segments, and revenue trends. The workflow includes data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis, and dashboard creation. The final insights are presented through a Power BI dashboard and a presentation created using Gamma.

## Dataset
The dataset contains customer transaction details such as:
- Age
- Purchase amount
- Product category
- Shipping type
- Previous purchases
- Customer demographics

## Tools & Technologies
- **Python:** Data cleaning and EDA (Pandas, Numpy)
- **SQL:** Querying and analysis (PostgreSQL)
- **Power BI:** Interactive dashboard creation
- **VS Code:** Analysis environment

## Project Steps
1. Loaded the dataset using Python.
2. Performed Exploratory Data Analysis (EDA).
3. Cleaned and prepared the data.
4. Ran SQL queries to analyze:
   - Customer segments
   - Revenue trends
   - Age-based spending
   - Shipping preferences
5. Built an interactive Power BI dashboard.

## Dashboard Highlights
- Customer segmentation (New, Returning, Loyal)
- Revenue by age group
- Category-wise sales analysis
- Shipping method comparison
- Purchase behavior trends

## Key Results
- Loyal customers contributed the highest share of revenue.
- Certain age groups showed higher spending patterns.
- Standard shipping was the most frequently used option.
- A few product categories generated the majority of sales.
